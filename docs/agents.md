# APP - WEB catalogo

## Contexto del proyecto
Este es un proyecto para desarrollar una web App sencilla que permite la visualización de un pdf específico en linea. La idea del proyecto es poder generar una instaccia virtual para acceder a un catalogo específico diseñado por el cliente el cual ya se encuentra en formato PDF.

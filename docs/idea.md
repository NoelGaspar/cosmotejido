# APP web - Catalogo

Este es un proyecto para desarrollar una web App sencilla que permite la visualización de un pdf específico en linea. La idea del proyecto es poder generar una instancia virtual para acceder a un catalogo específico diseñado por el cliente el cual ya se encuentra en formato PDF.

## Función principal

El proyecto considera un sitio web compuesto de dos páginas. la primera contiene una descripción sencilla del proyecto general y un boton que lleva a la segunda página. La segunda página contiene solo un visualizador de un PDF específico (fijo). La idea es que este visualizador permita leer el pdf página a página.

El sitio debe contar con un footer que tenga información sobre la empresa que financió el proyecto, y las redes sociales.

El sitio no necesita un header. sin embargo sería ideal contar con uno sencillos.

La pagina inicial debe contener una imagen un poco de texto y un boton. tipo call to action.
